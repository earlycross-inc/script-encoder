# script-encoder

Correct the encoding and line feed code of the script

## Install

Add the following line to manifest.json

```json
"com.earlycross.script-encoder": "https://github.com/earlycross-inc/script-encoder.git",
```

Example:

```json
{
  "dependencies": {
    "com.earlycross.script-encoder": "https://github.com/earlycross-inc/script-encoder.git",
    ...
    }
 }
```
